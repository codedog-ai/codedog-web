#title
你的AI Code Review伙伴

#description
Codedog 使用 [OpenAI GPT](https://www.openai.com) 帮助分析审查PR代码变动。我们的目标是让Code Review变的更可行、高效与舒适

#extra
  ::list
  - Codedog会帮你总结PR的改动内容，区分功能改动和代码格式化
  - 提出一些代码修改的建议
  - 支持[Github](https://www.github.com)，并且很快将会支持[Gitlab](https://www.gitlab.com)
  - 2分钟内即可完成安装
  ::
::



## 试用Codedog :badge[v0.0.3 Alpha]

在你的Github项目中添加Pull Request Webhook:
```plain
https://api.codedog.ai/v1/webhook/github
```

*Codedog 目前无法在限制PR评论权限的项目之中使用，我们在Beta版本中会升级到Github APP以解决这个问题（查看 [Roadmap](/roadmap)）*
